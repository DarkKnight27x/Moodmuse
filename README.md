# MoodMuse 🎵

**AI-Powered Music Recommendation Platform**

Discover new music that truly understands *you* — based on your personality, mood, visual preferences, and listening behavior.

###  Features
- Personality psychology assessment (10 weighted questions)
- Visual preference test (image-based)
- Unique **Music DNA** profile with beautiful visualization
- Hybrid recommendation engine (Personality + Mood + History)
- AI Music Therapist
- Monthly music psychology reports
- Relationship compatibility analysis
- Spotify integration (playlists & playback)

### 🛠 Tech Stack
- **Frontend**: React + TypeScript + Vite + Tailwind CSS + Framer Motion + Zustand
- **Backend**: FastAPI (Python) + MongoDB
- **Auth**: Google, Email, Spotify OAuth
- **Music API**: Spotify Web API

###  Project Status
Full-stack portfolio project in active development.

**Goal**: Build a production-grade AI music platform to showcase end-to-end skills.
